Electoral address uploader

See the overview for a description of electoral address uploading.

See the wiki FAQ for common questions and answers.

To start working on an electoral address job in QGIS you can either start a new QGIS project, open the electoral address upload management window (below), and select a new or existing job to work on. Alternatively you can open QGIS project in which you have saved an electoral address job.

You do not need to save the QGIS project to save the electoral address update - that is saved in the underlying database as you work on it.

Uploader tools

The electoral address uploader provides three tools for working with address data sets. These can be accessed from the Plugins | Electoral_address menu item, or from the Electoral address toolbar.

Management window

This is the main tool for managing address uploads. It is used load new datasets, select data sets to edit, edit the supplied address information, define how each address will be used, and generate Landonline upload files.

There is more detailed information on how to use this tool

When a new address data set is loaded the address layer is loaded into Qgis. Also Landonline address point, street, and parcel layers are loaded. (Note that the landonline layers only display at larger scales).

Address info tool

This is a map tool used to get information from the map about addresses. When it is selected an information window appears which displays information about uploaded addresses, landonline address points, and landonline streets. Select this tool then click or drag a rectangle on the map to get information. The tool remains active until another map tool (eg the QGis info tool) is selected.

Address highlighter

When working with electoral addresses there is often a "current address" that is being worked on. This tool automatically highlights that address on the map. If the landonline street and address layers are showing, then the linked street and address are also highlighted.

The current uploaded address is highlighted with a square, the linked landonline address point (if any) with a cross, and the linked landonline street with a line.

The highlighter button toggles this highlighting on and off

Landonline address deleting tool

Use this tool to mark Landonline addresses for deletion when there is not a delete record supplied with the job. Select this tool and then click on a Landonline address to delete it. This creates a new "delete" record linked to the landonline record.

Create address geometry tool

Use this tool to create or move the geometry for an address. Select this tool and then click on the map canvas to create a point geometry for the selected address. The map and address warnings will refresh and show the new address geometry. Click again to move the point. The tool remains active until another map tool is selected. This creates a geometry for the address in the postgres database.
